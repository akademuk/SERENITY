# SERENITY BAY — реміум шаблон для курорту та спа

ей проект — це втілення спокою, розкоші та єднання з природою. еб-сайт створено для презентації елітних курортів, готелів, спа-комплексів або ретрит-центрів.

## Що отримує клієнт?

- **тмосферний дизайн:** икористання текстур (шум, патерни), вишуканої типографіки та "повітря" у макеті передає відчуття релаксу та ексклюзивності.
- **анурення:**
  - **лавна прокрутка (Lenis):** Сайт рухається м'яко, наче хвилі, створюючи кінематографічний ефект.
  - **Інтерактивна карта:** нікальний блок з картою острова/курорту та інтерактивними точками інтересу.
  - **ідео-фони:** иві зображення, що передають атмосферу місця.
- **онтентні блоки:**
  - **езиденції:** етальний опис вілл та номерів.
  - **астрономія:** резентація ресторанів та меню.
  - **итуали:** озклад спа-процедур та подій.
  - **урнал (Editorial):** лок для новин, історій та анонсів.
- **ронювання:** Стильна та зручна форма для запиту на резервацію.

## ніверсальність та Теми

Шаблон розроблено так, щоб підкреслити унікальність вашої локації. будовані теми дозволяють адаптувати дизайн під час доби або настрій бренду.

**будовані теми:**

1.  **Midnight (Default):** Таємнича, вечірня атмосфера. Темні тони, золото, інтимність.
2.  **Sunrise:** Свіжість ранку. Світлі, теплі відтінки, відчуття початку нового дня.
3.  **Ocean:** либокі бірюзові та сині кольори. кцент на воді, морі та свіжості.

Також доступне перемикання шрифтів між класичними (Serif) та сучасними (Mono/Sans) варіантами.

## Технології

- **HTML5 / CSS3:** Сучасна верстка, адаптивність під усі пристрої.
- **JavaScript (ES6+):** одульна структура.
- **GSAP & ScrollTrigger:** Складні анімації появи елементів та паралакс ефекти.
- **Lenis:** реміальна плавна прокрутка.
- **SplitType:** німація тексту (поява по літерах/словах).
