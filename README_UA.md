# SERENITY BAY — Преміум шаблон для курорту та спа

Цей проект — це втілення спокою, розкоші та єднання з природою. Веб-сайт створено для презентації елітних курортів, готелів, спа-комплексів або ретрит-центрів.

## Що отримує клієнт?

- **Атмосферний дизайн:** Використання текстур (шум, патерни), вишуканої типографіки та "повітря" у макеті передає відчуття релаксу та ексклюзивності.
- **Занурення:**
  - **Плавна прокрутка (Lenis):** Сайт рухається м'яко, наче хвилі, створюючи кінематографічний ефект.
  - **Інтерактивна карта:** Унікальний блок з картою острова/курорту та інтерактивними точками інтересу.
  - **Відео-фони:** Живі зображення, що передають атмосферу місця.
- **Контентні блоки:**
  - **Резиденції:** Детальний опис вілл та номерів.
  - **Гастрономія:** Презентація ресторанів та меню.
  - **Ритуали:** Розклад спа-процедур та подій.
  - **Журнал (Editorial):** Блок для новин, історій та анонсів.
- **Бронювання:** Стильна та зручна форма для запиту на резервацію.

## Універсальність та Теми

Шаблон розроблено так, щоб підкреслити унікальність вашої локації. Вбудовані теми дозволяють адаптувати дизайн під час доби або настрій бренду.

**Вбудовані теми:**

1.  **Midnight (Default):** Таємнича, вечірня атмосфера. Темні тони, золото, інтимність.
2.  **Sunrise:** Свіжість ранку. Світлі, теплі відтінки, відчуття початку нового дня.
3.  **Ocean:** Глибокі бірюзові та сині кольори. Акцент на воді, морі та свіжості.

Також доступне перемикання шрифтів між класичними (Serif) та сучасними (Mono/Sans) варіантами.

## Технології

- **HTML5 / CSS3:** Сучасна верстка, адаптивність під усі пристрої.
- **JavaScript (ES6+):** Модульна структура.
- **GSAP & ScrollTrigger:** Складні анімації появи елементів та паралакс ефекти.
- **Lenis:** Преміальна плавна прокрутка.
- **SplitType:** Анімація тексту (поява по літерах/словах).

Цей сайт не просто інформує — він занурює гостя в атмосферу вашого курорту ще до приїзду.
